操作方法
・マウスの左クリックでセルを開く

実装方法
スタート時に、cellStateがすべてNoneのセルを置く。
どこかがクリックされたら、そのセルが開かれているかのbool値をtrueにする。
そして、クリックされたものを除く用意されているすべてのセルに対して、
ランダムに地雷を配置し、その後それぞれのセルから周囲8セルに対し地雷があるか確認する関数を呼ぶ。
クリックされたセルがNoneだったら、そのセルから周囲８セルに対して
クリックされたときと同じ処理をする関数を作る。
地雷が設定されているセルをクリックしてしまった場合は、すべてのセルを開ける関数を作り、ゲームオーバーとする。
